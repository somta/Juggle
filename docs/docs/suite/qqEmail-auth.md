#  QQ邮箱如何授权？
QQ邮箱是腾讯公司2002年推出，向用户提供安全、稳定、快速、便捷电子邮件服务的邮箱产品，已为超过1亿的邮箱用户提供免费和增值邮箱服务。 官网：https://mail.qq.com/

:::warning
注意: 如果需要大批量发送邮件，建议使用阿里云邮箱，因为QQ邮箱不支持批量邮件发送
:::
